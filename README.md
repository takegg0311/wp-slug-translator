# WordPress Slug Translator

ページタイトルからスラッグを付与時、英数字以外が含まれている場合は、内容を翻訳して英数字のスラッグを生成・付与するWordPressプラグインです。

## 概要

WordPress Slug Translatorは、ページや投稿のタイトルから自動的に英数字のスラッグを生成するWordPressプラグインです。タイトルに日本語などの非英数字文字が含まれている場合、大規模言語モデルAPIを使用して英語に翻訳し、URLスラッグに適した形式（小文字、ハイフン区切り）でスラッグを生成します。

## 主な機能

- **自動スラッグ生成**: ページ/投稿の保存・公開時に、スラッグが空または英数字以外が含まれている場合、自動的に翻訳してスラッグを生成
- **複数API対応**: OpenAI、Google Gemini、Anthropic Claudeの3つのAPIに対応
- **フォールバック機能**: 最大3つのAPIを優先順位で設定可能。1つ目のAPIが失敗した場合、自動的に2つ目、3つ目のAPIを試行
- **重複チェック**: 既存のスラッグと重複しないよう、自動的にユニークなスラッグを生成
- **カスタマイズ可能なプロンプト**: 翻訳用プロンプトを管理画面からカスタマイズ可能
- **ログ機能**: 翻訳処理の実行履歴を記録・参照可能（最大100件）
- **APIキー暗号化**: セキュリティのため、APIキーは暗号化して保存

## システム要件

- WordPress 5.0以上
- PHP 7.4以上
- 大規模言語モデルAPIのAPIキー（OpenAI、Google Gemini、Anthropic Claudeのいずれか）

## インストール方法

1. プラグインディレクトリに `wp-slug-translator` フォルダを配置
2. WordPress管理画面の「プラグイン」メニューからプラグインを有効化
3. 「設定」→「WordPress Slug Translator」から設定を開始

## 使用方法

### 初期設定

1. WordPress管理画面の「設定」→「WordPress Slug Translator」を開く
2. 「API設定」タブで、優先順位1〜3の順にAPIプロバイダー、APIキー、モデル名を設定
   - **対応API**: OpenAI、Google Gemini、Anthropic Claude
   - **デフォルトモデル**:
     - OpenAI: `gpt-4o-mini`
     - Google Gemini: `gemini-1.5-flash`
     - Anthropic Claude: `claude-3-5-sonnet-latest`
3. 「接続テスト」ボタンでAPI接続を確認
4. 「プロンプト設定」タブで翻訳用プロンプトをカスタマイズ（任意）
5. 設定を保存

### 動作タイミング

プラグインは以下のタイミングで自動的にスラッグを生成・更新します：

- **ページ保存時**: スラッグが付与されていない場合
- **投稿保存時**: スラッグが付与されていない場合
- **ページ公開時**: スラッグが付与されていない場合
- **投稿公開時**: スラッグが付与されていない場合
- **ページ編集時**: 既に付与されているスラッグが英数字のみで構成されていない場合
- **投稿編集時**: 既に付与されているスラッグが英数字のみで構成されていない場合

### スラッグ生成の流れ

1. タイトルからWordPress標準の方法でスラッグを生成
2. スラッグが英数字のみで構成されている場合、そのまま使用
3. スラッグに英数字以外が含まれている場合、APIでタイトルを翻訳
4. 翻訳結果をサニタイズ（英数字とハイフンのみ、小文字に変換）
5. 重複チェックを実行し、ユニークなスラッグを生成
6. スラッグを更新

### 管理画面機能

#### API設定タブ

- 最大3つのAPIを優先順位で設定
- 各APIの接続テスト機能
- APIキーの暗号化保存

#### プロンプト設定タブ

- 翻訳用プロンプトのカスタマイズ
- `{title}` プレースホルダーを使用してタイトルを挿入
- デフォルトプロンプトの復元機能

#### テスト機能タブ

- 指定したタイトルで翻訳処理をテスト
- 翻訳結果と使用したAPI、実行時間を確認

#### ログタブ

- 翻訳処理の実行履歴を表示（最大100件）
- タイトル、生成されたスラッグ、使用したAPI、実行時間、エラー情報を記録
- ログのクリア機能

## 技術仕様

### 対応API

- **OpenAI API**: Chat Completions APIを使用
- **Google Gemini API**: Generative Language APIを使用
- **Anthropic Claude API**: Messages APIを使用

### スラッグサニタイズ

生成されたスラッグは以下の処理を経てサニタイズされます：

- 改行や余分な空白を削除
- 英数字とハイフンのみを許可
- 複数のハイフンを1つに統合
- 先頭・末尾のハイフンを削除
- 小文字に変換

### APIキー暗号化

APIキーはWordPressの `wp_salt()` を使用して暗号化され、データベースに保存されます。
