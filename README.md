# WordPress Slug Translator

ページタイトルからスラッグを付与時、英数字以外が含まれている場合は、内容を翻訳して英数字のスラッグを生成・付与するWordPressプラグインです。

## システム要件

- 大規模言語モデルAPIへ接続する
  - OpenAI API / Gemini API / Claude を指定可能
  - WordPress管理画面上でAPIを入力する
  - 複数モデルに対応し、1つ目のモデルで失敗した場合は、2つ目のモデルで再試行する (最大3つまで実行順番を設定)
- スラッグが英数字のみで構成されていない場合、ページタイトルを翻訳して簡潔な英数字スラッグを生成する
  - 翻訳用プロンプトは管理画面から指定できる
  - 既に付与されているスラッグと重複が無いかも確認する
- 変換処理を行うタイミングは以下の通り
  - ページ保存時に、スラッグが付与されていない場合
  - 投稿公開時に、スラッグが付与されていない場合
  - ページ編集時に、既に付与されているスラッグが英数字のみで構成されていない場合
  - 投稿編集時に、既に付与されているスラッグが英数字のみで構成されていない場合

## 管理画面機能

WordPress上の管理画面で、以下の機能を設定する

- 言語モデルAPI Keyの設定
- 翻訳用プロンプトの設定
- 翻訳処理のテスト機能
- 翻訳実行ログの参照
